# url-shortener
A simple and fast url shortening service made in Golang.

<p>
  This program is written in Golang. The api's are created using go-fiber. Redis is used to provide fast data storage and retrival.
  This also used uses docker to run the application in its separate environment in localhost.
</p>

<hr/>
To start the program just copy clone the repo to your local machine. cd into the parent directory of docker compose and run

```sh

docker-compose up -d

```
